Basic Concepts

Core ideas behind linear regression, explained simply

01 — What is Linear Regression?

Linear Regression is a machine learning algorithm that finds the relationship between one or more input variables and a numerical output, by fitting a straight line through the data.

Given enough examples, the model learns to predict new values it has never seen before — such as estimating a person's weight based on their height, age, and habits.

02 — Independent and Dependent Variables

Independent Variable (X)

The input the model receives to make a prediction.

e.g. Height, Age, Physical activity

Dependent Variable (Y)

The output the model tries to predict — it depends on X.

e.g. Body weight in kg

03 — Training Data

Training data is the set of labeled examples used to teach the model. Each record has known input values (X) and a known correct output (Y). The algorithm studies these examples to find patterns.

  • Training set (80%): used to fit the model and learn the coefficients.
  • Test set (20%): held out to evaluate how well the model performs on new data.

04 — The Regression Line

The regression line is the straight line that best fits all the data points. The algorithm positions it to minimize the distance between each point and the line — this method is called Ordinary Least Squares (OLS).

05 — Slope and Intercept

y  =  β₀  +  β₁x

y — predicted value β₀ — intercept β₁ — slope x — input variable

β₀

Intercept

The predicted value of Y when X is zero. It is the point where the line crosses the Y-axis.

β₁

Slope

How much Y changes for each one-unit increase in X. A positive slope means Y grows as X grows.

06 — Example of Application

Using Height alone to predict Weight, the model learns an equation like:

Weight = −105.4 + 112.8 × Height

For a person with a height of 1.75 m:

Weight = −105.4 + 112.8 × 1.75 = 92.0 kg

Input (X)

Height = 1.75 m

Calculation

−105.4 + 112.8 × 1.75

Predicted output (Y)

92.0 kg

Our project extends this idea to 15 input variables simultaneously — age, height, eating habits, physical activity, and more — making it a Multiple Linear Regression model.
